#### 自己简单手写的全栈项目
> 前端页面由vite加vue3构建，nginx部署托管，
> 后端接口使用node+express构建，数据库用的mongodb
> 此代码仓库包含前端、后端、后台管理端的增删改查，注册登陆，所有服务均部署在unbutu下的docker容器中

**目录结构**：
> **web** --前端页面，
> **backstage**--后台录入管理端，
>> 两者都是使用vite构建，基于vue3+elementplus，使用pinia进行状态管理；
> **interface**--后端接口,文件上传操作使用multer。
